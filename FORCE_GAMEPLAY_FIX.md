# HUD de jogabilidade forçado

Este pacote coloca o HUD logo após a abertura do `<body>`, com CSS e JS próprios, usando `z-index` máximo.

Ele aparece mesmo se o resto da página der erro.

Suba principalmente:

- `index.html`
- `matsuri.html`
- `FORCE_GAMEPLAY_FIX.md`

Depois atualize o site com Ctrl+F5 ou abra em aba anônima.
