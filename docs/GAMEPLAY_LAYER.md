# Camada de Jogabilidade adicionada

Este pacote adiciona uma jogabilidade MVP à versão standalone HTML:

- HUD com XP, moedas, palavras e streak
- Missões clicáveis
- Trilha de progressão do Reino 1
- Biblioteca viva de vocabulário
- Diálogo básico com Sifu Chen
- Save local via localStorage
- Botão de trilha sonora ambiente já existente no canto inferior direito

Arquivo principal para GitHub Pages emergencial:

```txt
index.html
```

Também foi aplicado em:

```txt
matsuri.html
```
